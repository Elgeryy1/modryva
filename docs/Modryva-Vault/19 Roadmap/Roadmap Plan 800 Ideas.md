# Roadmap — Plan 800 Ideas

## Estado
**planned (marco de priorización).** Banco de ~800 ideas destiladas a **12 motores** + **19 épicas** +
**roadmap de 8 fases**, documentado en `docs/PLAN-800-IDEAS.md`.

## Tesis
**Extender lo ya desplegado, no construir de cero.** Prioriza quick-wins que reusan datos que ya se escriben
(`activity-log` + [[Modelo ChatSetting]]) con 0 tablas nuevas, y descarta lo que choca con "muros de datos"
(requiere infraestructura que no existe). Esto conecta directamente con [[Roadmap Wiring Idea Bank]] (cablear
lógica pura ya escrita) y con la lista de features `partial` del Vault.

## Cómo usarlo
Es el **filtro estratégico** para decidir qué feature del [[Idea bank|banco de ideas]] cablear a
continuación: alto valor percibido / bajo coste marginal primero.
