## Backups, import/export y migraciones

### 54. Backups automaticos

Backups diarios/semanales:

- Config de grupo.
- Red.
- Rutas.
- Plantillas.
- Automations.
- Bienvenidas/reglas.

Restore por fecha.

### 55. Snapshots antes de cambios grandes

Antes de:

- Aplicar plantilla.
- Unir red.
- Cambiar rutas.
- Activar membership comun.
- Importar backup.

Guardar snapshot y permitir rollback.

### 56. Comparar grupos

Vista diff:

- Grupo A tiene captcha off.
- Grupo B tiene logs a staff.
- Grupo C tiene reglas globales.

Accion: copiar solo diferencias seleccionadas.

### 57. Import desde otros bots

Importar desde:

- CSV/manual.
- Export propio.
- Otros bots si el formato se puede adaptar.

Objetivo: migrar comunidades a Modryva sin empezar de cero.
