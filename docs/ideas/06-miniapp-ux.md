## MiniApp y UX

### 41. Onboarding guiado

Cuando el bot entra a un grupo:

1. Comprobar permisos.
2. Elegir plantilla.
3. Configurar logs/staff.
4. Unir a red si aplica.
5. Crear backup inicial.

### 42. Modo demo

MiniApp con datos falsos para ensenar Modryva:

- Grupos demo.
- Alertas demo.
- Usuarios demo.
- Bots hijos demo.
- Sin tocar produccion.

### 43. Ayuda contextual

Ayuda por pantalla:

- Tooltips.
- Ejemplos.
- "Que hace esto".
- "Impacto al activarlo".

Sin llenar la UI de texto tecnico.

### 44. Builder visual de mensajes

Para bienvenidas, reglas, anuncios y posts:

- Preview.
- Botones.
- Variables.
- Idiomas.
- Envio programado.
- Enviar como bot hijo.

### 45. Variables avanzadas

Variables para plantillas:

- `{user_level}`
- `{group_name}`
- `{rules_link}`
- `{invite_count}`
- `{network_name}`
- `{bot_name}`
- `{user_badges}`

### 46. Multiidioma real

Extender i18n a:

- Bot.
- MiniApp.
- Mensajes de moderacion.
- Casino/juegos.
- Plantillas.
- D1.
- Owner panel.

### 47. Tono del bot

Cada grupo/bot hijo puede elegir tono:

- Serio.
- Casual.
- Gamer.
- Corporativo.
- Muy corto.
